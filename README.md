# Pantry to Plate
## Setup
1. Clone the repository: git clone https://github.com/yourusername/pantry-to-plate.git
2. Install dependencies:
   npm install
4. Create a '.env' file in the project root (based on the .env.example) and add your own OPENAI API key:
   OPENAI_API_KEY = your_api_key_here
6. Run server :
   node server.js
